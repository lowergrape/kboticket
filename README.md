# ⚾ BaseAlert — 프로야구 빈자리(취소표) 실시간 알림 서비스

인기 프로야구 경기는 순식간에 매진되지만, **취소표(빈자리)**는 수시로 발생합니다.
BaseAlert는 원하는 **경기 · 좌석 구역 · 희망 가격대**를 등록해두면,
빈자리가 생기는 순간 **실시간으로 알림**을 보내주는 웹 서비스입니다.

> **데모로 바로 보기:** 첫 화면의 **🚀 데모 계정으로 둘러보기** 버튼을 누르면
> 샘플 알림이 등록된 상태로 시작되어, 실시간 감시·알림이 작동하는 모습을 바로 볼 수 있습니다.

---

## ✨ 주요 기능 (PRD 반영)

| 기능 | 설명 |
|---|---|
| 🔐 회원가입 / 로그인 | 이메일·닉네임·비밀번호 기반 가입 및 로그인, 로그인 상태 유지 (기능 1) |
| 🔍 경기 검색 | 응원 팀 · 날짜 · 경기장으로 KBO 경기 검색 (기능 2) |
| 💺 좌석 현황 조회 | 구역별 가격 · 판매중/매진 상태 · 잔여 좌석 수 표시 (기능 3) |
| 🔔 빈자리 알림 설정 | 경기 · 좌석 구역 · 희망 최대 가격을 지정해 감시 등록 (기능 4) |
| 📡 실시간 좌석 감시 | 좌석 현황을 주기적으로 확인하고 변화를 감지 (기능 5) |
| ⚡ 빈자리 알림 발송 | 조건에 맞는 빈자리 발생 시 즉시 알림 + 기록 저장 (기능 6) |

## 🛠 기술 스택

- **프론트엔드:** HTML5, CSS3(반응형), JavaScript (외부 빌드 도구 없음)
- **데이터 저장:** 브라우저 `localStorage`
  - 서버/DB 없이 동작하도록, PRD의 데이터 관리(CSV/딕셔너리) 역할을 `localStorage`가 대신합니다.
  - 데이터는 **사용한 브라우저에만** 저장됩니다.

### ⚠️ 실시간 좌석 데이터에 대하여 (중요)
실제 예매 사이트(티켓링크 등)는 외부에서 좌석 데이터를 가져올 수 있는 공개 API를 제공하지 않습니다.
따라서 이 데모에서는 **좌석 현황이 시간에 따라 변하는 과정을 시뮬레이션**합니다.
(매진 구역에서 취소표가 발생하거나, 판매 중 좌석이 마감되는 상황을 무작위로 재현)
→ 감시 중인 구역에 빈자리가 생기면, 실제 서비스와 동일하게 **즉시 알림이 발송**됩니다.
실제 상용화 시에는 이 부분을 예매처와의 제휴 API 또는 웹 크롤링 모듈로 대체하게 됩니다.

## 🚀 GitHub Pages로 배포하기 (무료 인터넷 주소 만들기)

### 방법 A. 웹 업로드 (가장 쉬움, 추천)

1. [github.com](https://github.com) 가입 후 로그인
2. 오른쪽 위 **`+` → New repository** 클릭
3. **Repository name** 에 `basealert` 입력 → **Public** 선택 → **Create repository**
4. 새 저장소 화면에서 **uploading an existing file** 링크 클릭
5. 이 폴더 안의 **`index.html`, `css` 폴더, `js` 폴더, `README.md`** 를 한꺼번에 끌어다 놓기
   > ⚠️ `basealert` 폴더째로 올리지 말고, **폴더 안의 내용물**을 올려야 `index.html`이 저장소 맨 위에 옵니다.
6. 아래 **Commit changes** 클릭
7. 상단 **Settings → 왼쪽 메뉴 Pages** 이동
8. **Source** 를 `Deploy from a branch`, **Branch** 를 `main` / `/(root)` 로 선택 → **Save**
9. 1~3분 뒤 같은 Pages 화면 맨 위에 나타나는 주소가 완성된 사이트입니다:
   `https://<내 깃허브아이디>.github.io/basealert/`

→ **이 주소를 과제로 제출**하면 됩니다.

### 방법 B. git 명령어로 올리기

이 폴더에는 이미 git 저장소가 초기화되어 있고 첫 커밋이 되어 있습니다.
GitHub에서 빈 저장소(`basealert`)를 만든 뒤, 아래만 실행하세요.

```bash
git remote add origin https://github.com/<내 깃허브아이디>/basealert.git
git branch -M main
git push -u origin main
```

이후 GitHub의 **Settings → Pages** 에서 위 8~9번과 동일하게 설정합니다.
